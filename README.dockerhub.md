# Larios Income Tax Website

Angular-based web application for Larios Income Tax and Immigration services.

## Overview

This is the official website for Larios Income Tax, providing comprehensive tax preparation, immigration services, and translation services to clients in San Diego, California.

## Features

- Modern Angular 21 application with standalone components
- Angular Material Design implementation with custom theming
- Responsive design optimized for desktop, tablet, and mobile devices
- Service information and booking functionality
- Contact forms for United States and Mexico locations
- Multi-language support preparation

## Services Offered

### Tax Preparation

Professional tax preparation services with free consultation. Duration: 50 minutes per session.

### Immigration Services

Comprehensive immigration assistance and documentation services. Duration: 50 minutes per session.

### Translations

Professional translation services for various documents and materials. Duration: 1 hour 50 minutes per session.

## Technology Stack

- Frontend Framework: Angular 21
- UI Library: Angular Material Design
- Language: TypeScript
- Styling: CSS with custom Material theme
- Build Tool: Angular CLI with esbuild

## Container Information

This Docker image contains a production-ready build of the Larios Income Tax website, served through Nginx. The application is optimized for performance with lazy-loaded routes and efficient bundle sizes.

### Available Platforms

- linux/amd64
- linux/arm64

### Port Configuration

The application runs on port 80 inside the container.
